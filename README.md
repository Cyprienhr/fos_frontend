
## How to navigate to features

 - Admin Dashboard: open `/admin-dashboard` after logging in as admin
 - Approve/Decline an order: open `/admin-dashboard` → Manage Orders → click **Review** on a pending order → Approve/Decline
 - Add fertilizer: open `/admin-dashboard` → Fertilizer Rates → fill the Add form → Submit
 - Farmer place order: open `/farmer-dashboard` → New Order → select fertilizer → Submit → switch to My Orders to view
